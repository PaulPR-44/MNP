# Compte rendu Projet MNP S1
Binôme: Paul PERON-REDON et Nicolas FORTUN
## Choix du sujet

Le projet que nous avons choisi de modéliser est le problème à 3 corps. Plus précisement, nous répondrons à la problématique: **Comment trouver des condition initiales afin de satisfaire la stabilité du problème à 3 corps ?**

Nous choisirons de nous positionner dans un système planétaire, type Soleil-Terre-Saturne.

Nous nous appuyerons sur les équations différentielles définies sur la page Wikipédia: [Le problème à N corps](https://fr.wikipedia.org/wiki/Probl%C3%A8me_%C3%A0_N_corps)

Pour cela, l'algorithme mis en place sera celui de la méthode de Runge-Kutta d'ordre 4.

## Conditions

Nous simulerons un environnement en 2 dimensions via la librairie python *matplotlib*

Si le système parvient à une collision, nous fusionnerons les deux corps
